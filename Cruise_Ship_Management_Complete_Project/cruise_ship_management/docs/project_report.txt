Objective:
Develop an online system for voyagers to book cruise services and order items.

Frontend:
HTML, CSS, JavaScript

Backend:
Node.js + Express.js

Database:
Firebase

Features:

- Catering Orders
- Stationery Orders
- Beauty Salon Booking
- Fitness Center Booking
- Party Hall Booking

Optimization:
- Modular Code Structure
- Logging Support
- Maintainable Architecture
