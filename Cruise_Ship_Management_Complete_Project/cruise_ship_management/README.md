
# Cruise Ship Management System

## Technologies Used
- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- Firebase

## Modules
### Voyager
- Login
- Order Catering Items
- Order Stationery Items
- Book Beauty Salon
- Book Fitness Center
- Book Party Hall

### Admin
- Add/Edit/Delete Items
- Manage Voyagers

### Manager
- View All Bookings

### Head Cook
- View Catering Orders

### Supervisor
- View Stationery Orders

## How to Run

### Frontend
Open frontend/index.html in browser

### Backend
1. Open terminal
2. Run:
npm install
npm start

Backend runs on:
http://localhost:5000

## Database
Add your Firebase credentials inside:
database/firebase-config.js
